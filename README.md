# Superstore Sales Performance Dashboard

## Project Overview

This project presents an interactive Tableau dashboard built using the Sample Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, product categories, and top-performing products through dynamic visualizations and filters.

The objective of this project is to help business stakeholders monitor key performance indicators (KPIs), identify sales trends, analyze customer behavior, and make data-driven decisions.

---

## Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Total Orders
- Profit Margin %

### Interactive Analysis
- Sales by Category
- Sales by Segment
- Monthly Sales Trend
- Top 10 Products by Selected Metric

### Dynamic Features
- Measure Selector Parameter
  - Sales
  - Profit
  - Quantity

### Interactive Filters
- Region
- Category
- Segment

### Dashboard Actions
- Cross-filtering between visualizations
- Interactive exploration of business metrics

## Dataset

**Dataset:** Sample Superstore Dataset

The dataset contains retail transaction data including:
- Orders
- Sales
- Profit
- Product Information
- Customer Segments
- Regions
- Categories

Total Records Analyzed: **9,994**

---

## Key Insights

- Technology generated the highest sales among all product categories.
- Consumer segment contributed the largest share of revenue.
- Sales showed significant growth during the final quarter of the year.
- A small number of products contributed a large portion of total sales.
- Overall profit margin was approximately **12.47%**.
